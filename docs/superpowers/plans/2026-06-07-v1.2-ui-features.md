# v1.2 UI Features Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add four UI features to ExamDump v1.2 — a launch splash screen, an always-visible bottom status bar with a live clock, a drag-to-resize AI/browser panel, and an in-app changelog.

**Architecture:** Splash and panel sizing live in the Electron main process; the clock, drag handle, and changelog are renderer-side. Two pure helpers (`clampRatio`, `formatClock`) are extracted and unit-tested; the window/native-view behaviour is verified by running the app.

**Tech Stack:** Electron (main + preload), React + TypeScript (renderer), Vitest (`npx vitest run <file>`). The `RESIZE_PANEL` IPC channel already exists in `src/ipc/channels.ts`.

**Native module note:** if a Vitest run fails with a better-sqlite3 `NODE_MODULE_VERSION` mismatch, run `npm rebuild better-sqlite3` first (the helper tests here don't touch the DB, but the full suite does).

export const DEFAULT_RATIO = 0.6;
export const MIN_RATIO = 0.25;
export const MAX_RATIO = 0.75;

export function clampRatio(ratio: number): number {

}

import React, { useState, useEffect } from 'react';

export function formatClock(d: Date): string {
  const date = d.toLocaleDateString('en-US', { weekday: 'short', month: 'short', day: 'numeric', year: 'numeric' });
  const time = d.toLocaleTimeString('en-US', { hour: 'numeric', minute: '2-digit', second: '2-digit', hour12: true });
  return `${date} · ${time}`;
}

export function StatusBar() {
  const [now, setNow] = useState(() => new Date());
  useEffect(() => {
    const id = setInterval(() => setNow(new Date()), 1000);
    return () => clearInterval(id);
  }, []);
  return (
    <div className="status-bar">
      <span className="status-clock">{formatClock(now)}</span>
    </div>
  );
}
```

  const onDividerDown = (e: React.MouseEvent) => {
    e.preventDefault();
    draggingRef.current = true;
    const onMove = (ev: MouseEvent) => {
      if (!draggingRef.current) return;
      const ratio = (window.innerWidth - ev.clientX) / window.innerWidth;
      const clamped = Math.min(0.75, Math.max(0.25, ratio));
      setDragRatio(clamped);
      if (rafRef.current == null) {
        rafRef.current = requestAnimationFrame(() => {
          rafRef.current = null;
          window.electronAPI.resizePanel(clamped);
        });
      }
    };
    const onUp = () => {
      draggingRef.current = false;
      window.removeEventListener('mousemove', onMove);
      window.removeEventListener('mouseup', onUp);
    };
    window.addEventListener('mousemove', onMove);
    window.addEventListener('mouseup', onUp);
  };
```
